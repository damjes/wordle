# Wordle

Reimplementacja [Wordle](https://www.nytimes.com/puzzles/wordle)
w
[TypeScript](https://www.typescriptlang.org/)

-   [React](https://react.dev/)
-   [Vite](https://vite.dev/).

Możesz zagrać [online](https://wordle.damj.es/).

## Uruchomienie

Standardowy Vite:

```bash
npm install
npm run dev
```

Można też zbudować stronę statyczną:

```bash
npm install
npm run build
```
